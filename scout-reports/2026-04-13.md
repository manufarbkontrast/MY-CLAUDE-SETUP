# GitHub Scout-Bericht -- 2026-04-13

## Zusammenfassung

14 neue/relevante Repositories gefunden in 6 Kategorien. Schwerpunkt diese Woche: Das Claude-Code-Ecosystem waechst explosiv (mehrere Repos mit 30k+ Sternen), OpenClaw dominiert die Agent-Frameworks, und im Finanzbereich etabliert sich WrenAI als fuehrendes Open-Source-GenBI-Tool.

## 1. Claude Code Skills & Agents

### 1.1 hesreallyhim/awesome-claude-code
- **URL:** https://github.com/hesreallyhim/awesome-claude-code
- **Sterne:** ~38.300
- **Kategorie:** Skill / Inspiration
- **Relevanz:** Hoch
- **Beschreibung:** Umfassende kuratierte Liste von Skills, Hooks, Slash-Commands, Agent-Orchestratoren, Anwendungen und Plugins fuer Claude Code. Aktuell die groesste Referenzsammlung im Ecosystem.
- **Nutzen:** Zentrale Quelle zum Entdecken neuer Skills und Plugins, die ins eigene Setup uebernommen werden koennen. Ergaenzt die bereits genutzten Quellen (affaan-m, wshobson, secondsky).

### 1.2 shanraisshan/claude-code-best-practice
- **URL:** https://github.com/shanraisshan/claude-code-best-practice
- **Sterne:** ~37.900
- **Kategorie:** Inspiration / Skill
- **Relevanz:** Hoch
- **Beschreibung:** Referenz-Implementierung fuer Claude Code Best Practices -- Skills, Subagents, Hooks und Commands als strukturiertes Beispielprojekt. Motto: "Practice made Claude perfect."
- **Nutzen:** Direkt vergleichbar mit dem eigenen Setup. Gute Quelle fuer Hook-Patterns und Agent-Konfigurationen, die noch fehlen koennten.

### 1.3 sickn33/antigravity-awesome-skills
- **URL:** https://github.com/sickn33/antigravity-awesome-skills
- **Sterne:** ~32.000
- **Kategorie:** Skill
- **Relevanz:** Hoch
- **Beschreibung:** Installierbare Bibliothek mit 1.400+ agentischen Skills fuer Claude Code, Cursor, Codex CLI, Gemini CLI und mehr. Inklusive Installer-CLI, Bundles und Workflows. V9.13.0 aktuell.
- **Nutzen:** Mit dem enthaltenen CLI (`npx antigravity install`) koennen Skills direkt installiert werden. Die Bundle-Funktion erlaubt thematische Skill-Pakete -- koennte den eigenen `prompt-optimizer` ergaenzen.

### 1.4 VoltAgent/awesome-agent-skills
- **URL:** https://github.com/VoltAgent/awesome-agent-skills
- **Sterne:** ~15.100
- **Kategorie:** Skill
- **Relevanz:** Mittel
- **Beschreibung:** 1.000+ Agent Skills von offiziellen Dev-Teams (Anthropic, Vercel, Stripe, Cloudflare, Netlify, Sentry, Expo, HuggingFace, Figma) und der Community. Kompatibel mit Claude Code, Codex, Gemini CLI, Cursor.
- **Nutzen:** Offizielle Skills von Plattform-Anbietern, die nicht in anderen Sammlungen enthalten sind. Besonders Stripe-, Cloudflare- und Sentry-Skills koennten fuer Fullstack-Projekte nuetzlich sein.

### 1.5 travisvn/awesome-claude-skills
- **URL:** https://github.com/travisvn/awesome-claude-skills

- **Kategorie:** Skill / Inspiration
- **Relevanz:** Mittel
- **Beschreibung:** Kuratierte Sammlung von Claude Skills mit Qualitaetsfilter (Mindest-Sternezahl fuer Aufnahme). Fokus auf gepruefte, hochwertige Skills.
- **Nutzen:** Qualitaetsgefilterte Alternative zu den grossen Sammlungen -- guter Einstiegspunkt fuer gepruefte Skills.

### 1.6 alirezarezvani/claude-skills
- **URL:** https://github.com/alirezarezvani/claude-skills
- **Sterne:** ~5.200
- **Kategorie:** Skill
- **Relevanz:** Mittel
- **Beschreibung:** 232+ Skills & Agent-Plugins fuer Engineering, Marketing, Product, Compliance und C-Level Advisory. Nicht nur Technik, sondern auch Business-Skills.
- **Nutzen:** Einzigartig durch Business- und Compliance-Skills (Finance, Marketing, C-Level). Ergaenzt das techniklastige eigene Setup um geschaeftliche Perspektiven.

## 2. AI Agent Frameworks

### 2.1 openclaw/openclaw
- **URL:** https://github.com/openclaw/openclaw
- **Sterne:** ~250.000
- **Kategorie:** Tool / Workflow
- **Relevanz:** Hoch
- **Beschreibung:** Self-hosted, Open-Source AI-Agent-Assistent. Laeuft auf eigener Hardware, unterstuetzt WhatsApp, Telegram, Discord und 12+ Messaging-Plattformen. Von 9k auf 250k Sterne in wenigen Monaten gewachsen -- aktuell das am schnellsten wachsende AI-Repo auf GitHub.
- **Nutzen:** Als self-hosted Agent-Plattform interessant fuer die Anbindung an Messaging-Kanaele. OpenClaw-RL ermoeglicht das Training personalisierter Agents aus natuerlichem Feedback.

## 3. MCP Server

### 3.1 K-Dense-AI/claude-skills-mcp
- **URL:** https://github.com/K-Dense-AI/claude-skills-mcp
- **Sterne:** ~364
- **Kategorie:** MCP Server
- **Relevanz:** Hoch
- **Beschreibung:** MCP-Server zum Suchen und Abrufen von Claude Agent Skills mittels Vektor-Suche. Leichtgewichtiger Proxy (~15 MB), startet in <5 Sekunden. Semantische Aehnlichkeitssuche ueber Skill-Repositories.
- **Nutzen:** Koennte als MCP-Server ins eigene Setup integriert werden, um die 461 vorhandenen Skills per Vektorsuche durchsuchbar zu machen -- deutlich besser als der aktuelle keyword-basierte `prompt-optimizer`.

### 3.2 makafeli/n8n-workflow-builder
- **URL:** https://github.com/makafeli/n8n-workflow-builder
- **Sterne:** ~469
- **Kategorie:** MCP Server / Workflow
- **Relevanz:** Mittel
- **Beschreibung:** MCP-Integration fuer n8n Workflow Automation. Verbindet Claude Desktop, ChatGPT und andere AI-Assistenten per Model Context Protocol mit n8n fuer natuerlichsprachige Workflow-Verwaltung.
- **Nutzen:** Ermoeglicht die Steuerung von n8n-Workflows direkt aus Claude heraus. Interessant in Kombination mit dem bestehenden n8n-Setup.

## 6. Finanzanalyse & Reporting

### 6.1 Canner/WrenAI
- **URL:** https://github.com/Canner/WrenAI
- **Sterne:** ~14.900
- **Kategorie:** Finanz-Tool / Tool
- **Relevanz:** Hoch
- **Beschreibung:** Open-Source GenBI-Agent mit Semantic Layer. Text-to-SQL, Text-to-Chart, unterstuetzt 12+ Datenquellen (PostgreSQL, BigQuery, Snowflake etc.) und beliebige LLMs (OpenAI, Claude, Gemini, Ollama). Agentic BI als Strategie fuer 2026.
- **Nutzen:** Perfekt fuer Finanz-Dashboards und KPI-Reporting. Kann mit dem vorhandenen dbhub-MCP-Server kombiniert werden fuer natuerlichsprachige Datenbankabfragen mit automatischer Visualisierung.

### 6.2 JerBouma/FinanceToolkit
- **URL:** https://github.com/JerBouma/FinanceToolkit
- **Sterne:** ~4.600
- **Kategorie:** Finanz-Tool
- **Relevanz:** Mittel
- **Beschreibung:** Python-Toolkit mit 150+ Finanzkennzahlen, Indikatoren und Performance-Messungen. Transparent und einfach zu nutzen.
- **Nutzen:** Kann als Backend fuer einen Claude-Finance-Agent dienen. 150+ vorgefertigte Kennzahlen ersparen eigene Implementierung.
